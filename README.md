I work at one frontier on purpose: teaching machines to perceive 3D space. Right now that means
**3D Gaussian Splatting** — the representation embodied AI is converging on for perception, mapping, and
simulation. I'd rather be early and deep on one wave than broad and late on ten. Code is how I think;
I build in public so the work speaks before I do.

## Currently

- **Building** — 3D Gaussian Splatting pipelines for robot perception: turning raw RGB into a scene a robot can reason over.
- **Reproducing** — landmark work in splat-based world models, documented end-to-end (notes + runnable code).
- **Reading the edge** — Gaussian-SLAM, splat-based synthetic data for VLA policies, embodied world models.
- **Translating** — frontier papers into one clear idea each, for people who don't have time to read them ([@paperlabs.studio](https://instagram.com/paperlabs.studio)).
